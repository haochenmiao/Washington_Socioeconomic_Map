Washington Social and Economic Conditions Map

A three-dimensional map of Social and Econimic Conditions in Washington State census tracts

Go to Map

Leavenworth, 2019 - Jonathan Yu

Map Description:

This website hosts an interactive web map that displays various social and economic indicators across Washington state census tracts. The three-dimensional map represents the values of a selected indicator, such as poverty rates, unemployment, or demographic diversity, using both the height and color of each tract’s representing polygon. The current indicator displayed on the map is shown in the legend and can be changed by selecting different options from the sidebar. Additional details about the social and economic conditions within a specific census tract can be accessed by clicking on that tract.

The map on this website was created for educational purposes only and aims to highlight the distribution of social and economic conditions across Washington state, along with the potential inequalities associated with these factors.

Data:

The data used in this map was gathered from various sources, including social and economic indicators from the Washington Geospatial Open Data Portal. The variables in the data were normalized using min-max normalization to facilitate easier comparison across different socioeconomic factors. Further information about the data, variables, and methodology can be found on the accompanying resources page. data page.
